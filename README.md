# GameJar

**GameJar** adalah aplikasi untuk memantau harga game sekaligus membantu pengguna menabung agar bisa membeli game yang diinginkan.

## Tentang Project

Sering melewatkan diskon game atau kesulitan mengatur uang untuk membeli game?
GameJar dibuat untuk menjawab masalah tersebut dengan menggabungkan fitur pemantauan harga dan sistem tabungan dalam satu aplikasi.

## Fitur Utama

* Monitoring harga game dari berbagai platform
* Notifikasi saat harga game turun atau sedang diskon
* Sistem tabungan per game
* Progress tabungan untuk setiap target pembelian
* Wishlist game

## Tujuan

GameJar bertujuan membantu pengguna agar:

* Lebih hemat dalam membeli game
* Lebih terencana dalam mengatur keuangan
* Menghindari pembelian impulsif

## Penutup

GameJar bukan sekadar aplikasi tracker harga, tetapi juga alat untuk membantu pengguna mencapai tujuan mereka dalam membeli game dengan cara yang lebih bijak.
